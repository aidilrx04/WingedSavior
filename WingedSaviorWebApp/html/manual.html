Welcome to the Winged Saviors manual

This manual provides instructions and guidelines for using the application effectively in various scenarios.

Getting Started

Before flying the application, ensure that you have read and understood the user manual provided with your drone. Familiarize yourself with the controls and safety precautions.

Create a new project

  1. Go to Home page
  2. Click on the Create Project dropdown
  3. Click on Create Project button. This will take you to create project page
  4. Enter Project Name, Project Area, and choose at least one Drone Detection
  5. After all information is filled, click Create Project button and the project will be created
  6. The will take you to Project Detail page

View created projects

  1. Go to Home page
  2. Click on Project List dropdown
  3. Click on Project List button to go to the project list page
  4. This page will display all projects created by you, click on any of them to display the information
  5. Click
    • Start button to start the project
    • View button to view project details

Start a project

  1. Click Start button in the project you want to start at Project List page
  2. Click
    • Start Icon(Red Icon): to start the project
    • Stop Icon(Green Icon): to stop the project
  3. Additionally, you can view the project timer and project as well as the last project run time.
